Layers

Import
Selected layer
Module
Re-roll keeps the position and size but rolls a new layout for this module. Inks override the global palette for this layer only.
Generation controls
Variant (the dice)
Each number is one fixed layout. Step through them deterministically — same number always gives the same result. Re-roll jumps to a random one.
Ratio & size it generates
Re-generates content to fill a new shape/resolution — a larger or wider area packs in more (or fewer) elements. The on-canvas display scale is preserved.
Text source
Only affects modules that render words (caption, header, table, HUD labels, etc.). With in order on, words are used top-to-bottom (great for labelling State graph nodes in sequence); off, they're shuffled.
Text
Image · dither
Reduces an imported image to a two-tone riso/print dither. Pixel size controls chunkiness; halftone draws newsprint dots. Light-area transparency lets the paper (or warm background) show through.
