Generative poem light sculpture

memory 6.jpg — 2025

LED matrices and steel sheets resurfacing poem fragments from Flickr titles—an installation about collective memory and the stories buried in shared archives.

Medium

LED panels · steel

Collaboration

Flickr Foundation

Focus

Algorithmic poetry from image titles

Concept

Created in collaboration with the Flickr Foundation, this piece uses generative algorithms to weave poetry from the vast archive of Flickr image titles. Each poem emerges as the code pairs and rearranges captions, letting the words of countless strangers reshape new stories.

The installation becomes a reflection of digital memory and collective authorship, inviting people to rediscover unseen poetry scattered across the web.
